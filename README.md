## Задание 1. Фильтрация кратных чисел в бинарном файле  
**Условие (вариант 9):**  
Получить в другом файле все компоненты исходного бинарного файла, кроме тех, которые кратны заданному числу \( k \).

### Класс `Task1`
**Поля (private):**
- `sourcePath_` – путь к исходному файлу
- `destPath_` – путь к результирующему файлу
- `k_` – число для проверки кратности

**Конструкторы:**
- `Task1(const std::string& source = "numbers1.bin", const std::string& dest = "filtered1.bin", int k = 3)` – с параметрами по умолчанию.

**Методы (public):**
- `void execute() override` – главный метод выполнения задания.
- `std::string getName() const override` – возвращает название задания.
- `int getNumber() const override` – возвращает номер задания (1).

**Вспомогательные методы (private):**
- `void fillBinaryFileWithRandomNumbers(int count)` – заполняет исходный файл случайными целыми числами от 1 до 100.
- `void printBinaryFileContents(const std::string& description) const` – выводит содержимое бинарного файла.
- `void filterMultiples()` – фильтрует числа, записывая в новый файл те, которые **не** кратны \( k \).

### Алгоритм работы кода
1. Генерация исходного бинарного файла с 20 случайными числами (1–100).
2. Вывод содержимого исходного файла.
3. Фильтрация: чтение чисел из исходного файла, проверка `num % k != 0`, запись подходящих в новый файл.
4. Вывод содержимого отфильтрованного файла и статистики (сколько чисел осталось).

## Задание 2. Матрица из бинарного файла и поиск столбца  
**Условие (вариант 9):**  
Скопировать элементы заданного бинарного файла в квадратную матрицу размером \( n \times n \) (если элементов недостаёт, заполнить оставшиеся нулями). Указать столбец (назвать его номер), где произведение элементов дальше всего от нуля.

### Внутренний класс `Matrix`
**Поля (private):**
- `data_` – двумерный вектор (`std::vector<std::vector<int>>`)
- `size_` – размер матрицы (количество строк/столбцов)

**Конструктор:**
- `Matrix(size_t n)` – создаёт матрицу \( n \times n \), заполненную нулями.

**Методы (public):**
- `void setValue(size_t i, size_t j, int value)` – установка элемента.
- `int getValue(size_t i, size_t j) const` – получение элемента.
- `size_t getSize() const` – возвращает размер.
- `int findColumnFarthestFromZero() const` – ищет номер столбца (с 1) с максимальным модулем произведения элементов.
- `void print(const std::string& description) const` – выводит матрицу.

### Класс `Task2`
**Поля (private):**
- `filePath_` – путь к бинарному файлу
- `matrixSize_` – размер матрицы

**Конструкторы:**
- `Task2(const std::string& filePath = "numbers2.bin", size_t matrixSize = 3)` – с параметрами по умолчанию.

**Методы (public):**
- `void execute() override` – выполнение задания.

**Вспомогательные методы (private):**
- `void fillFileForMatrix(int count)` – заполняет бинарный файл \( n^2 \) случайными числами.
- `std::unique_ptr<Matrix> createMatrixFromFile() const` – читает числа из файла и формирует матрицу.

### Алгоритм работы кода
1. Создание бинарного файла с \( n^2 \) случайными числами.
2. Вывод исходных чисел в удобном для восприятия виде (по \( n \) в строке).
3. Формирование матрицы из файла.
4. Вывод матрицы.
5. Поиск столбца с максимальным модулем произведения элементов.
6. Вывод номера найденного столбца (нумерация с 1).

## Задание 3. Поиск игрушек для трёхлетнего ребёнка  
**Условие (вариант 9):**  
Файл содержит сведения об игрушках: название игрушки, стоимость в рублях и возрастные границы (минимальный и максимальный возраст). Получить сведения о том, можно ли подобрать игрушку, любую, кроме мяча, подходящую ребёнку трёх лет.

### Класс `Toy`
**Поля (private):**
- `name_` – название игрушки
- `price_` – стоимость (в рублях, с двумя знаками после запятой)
- `minAge_` – минимальный возраст
- `maxAge_` – максимальный возраст

**Конструктор:**
- `Toy(const std::string& name, double price, int minAge, int maxAge)`

**Методы (public):**
- Геттеры для всех полей.
- `bool isSuitableForAge(int age) const` – проверяет, подходит ли игрушка для указанного возраста.
- `std::string toString() const` – возвращает строковое представление.

### Класс `Task3`
**Поля (private):**
- `filePath_` – путь к бинарному файлу с игрушками
- `toys_` – вектор объектов `Toy`, загруженных из файла

**Конструктор:**
- `Task3(const std::string& filePath = "toys.bin")`

**Методы (public):**
- `void execute() override` – выполнение задания.
- `bool findToyForThreeYearOld() const` – поиск подходящей игрушки (не мяч, возраст 3).

**Вспомогательные методы (private):**
- `void loadToys()` – загрузка игрушек из бинарного файла.
- `void saveToys() const` – сохранение вектора игрушек в файл.
- `void generateRandomToys(int count)` – генерация случайных игрушек.

### Алгоритм работы кода
1. Генерация 10 случайных игрушек (названия из предопределённого списка, случайные цена и возраст).
2. Вывод списка всех игрушек.
3. Поиск игрушки, удовлетворяющей условиям: название ≠ "Мяч", возрастной диапазон включает 3 года.
4. Вывод результата (да/нет) и, если найдена, информации о ней.

## Задание 6. Фильтрация строк без русских букв  
**Условие (вариант 6):**  
Переписать в другой текстовый файл строки, в которых нет русских букв.

### Класс `Task6`
**Поля (private):**
- `sourcePath_` – путь к исходному файлу
- `destPath_` – путь к результирующему файлу
- `lines_` – вектор строк исходного файла

**Конструктор:**
- `Task6(const std::string& source = "text6.txt", const std::string& dest = "filtered6.txt")`

**Методы (public):**
- `void execute() override` – выполнение задания.

**Вспомогательные методы (private):**
- `void loadLines()` – чтение всех строк из исходного файла.
- `void saveLines(const std::vector<std::string>& lines, const std::string& path) const` – запись строк в файл.
- `bool containsRussianLetters(const std::string& text) const` – проверка наличия русских букв (поиск байтов `0xD0` или `0xD1` в UTF-8).
- `void generateRandomText(int lineCount)` – создание случайного текста из заготовленных фраз.
- `void filterLinesWithoutRussian() const` – фильтрация строк и запись результата.

### Алгоритм работы кода
1. Генерация 15 строк из предопределённого набора (смесь английских и русских фраз).
2. Вывод исходного файла.
3. Фильтрация: проход по всем строкам, проверка `!containsRussianLetters(line)`.
4. Запись отобранных строк в новый файл.
5. Вывод отфильтрованного файла и статистики.
